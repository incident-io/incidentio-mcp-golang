# incident.io MCP Server

> [!WARNING]
>
> This was a prototype GoLang implementation of an MCP (Model Context Protocol) server for incident.io, providing comprehensive tools to interact with the incident.io API. Built following industry-standard Go project layout patterns.
>
> In March 2026, it was superseded by an [officially supported MCP Server](https://incident.io/changelog/remote-mcp-server).
>
> Read more and configure it here: https://docs.incident.io/ai/remote-mcp
>
> We're not maintaining or accepting further pull requests or changes to this repository going forward.
> 
> Thank you to everyone who provided feedback in the past. Please direct all future MCP related feedback to support@incident.io 🙏
